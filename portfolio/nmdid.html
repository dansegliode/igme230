NMD ID Package

This animation package was created for the New Media Design program at RIT. It includes a title sequence, animated bug, lower third, bumper, and end credits.

My idea was to take different elements of new media like ui, web, 3d, and motion, and have all the parts work together like a rube goldberg machine to make the logo, similar to how all our areas of focus and our hard work make new media what it is.
